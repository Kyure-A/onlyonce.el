* onlyonce.el
Package that executes a function that you want to execute only once the first time.

* How to Use

if you use leaf:
#+begin_src emacs-lisp
  (leaf onlyonce
    :config
    (onlyonce-add '(you-want-to-execute-only-once-hogefuga-function))
    (onlyonce-startup))
#+end_src
